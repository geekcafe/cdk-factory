# No 3rd Party References

This is an open-source project under the geek-cafe organization. Do not reference any 3rd party names that may be used to guide features and functionality. Do not include them in any generated code, documentation, specs, comments, or configuration files.

## Rules

- Use generic or geek-cafe-branded naming instead
- When referencing context from other workspace projects, use a generic term like "Acme" instead of the actual project or company name
